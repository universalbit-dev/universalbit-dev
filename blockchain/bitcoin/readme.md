#### Pruned Node
A pruned Bitcoin node is technically still a Bitcoin full node, however it prunes its local blockchain in order to save disk space.
License: CC (ASA 3U)

```
prune=550
​
maxconnections=8
listen=0
maxuploadtarget=144
​
checkblocks=1
checklevel=0
​
txindex=0
```
