3. **Setup Instructions**:
   - Clone the repository:
     ```bash
     git clone https://github.com/universalbit-dev/universalbit-dev/
     cd universalbit-dev/chess/
     npm i
     npm i pm2 -g
     pm2 start ecosystem.config.js
     ```
